# signe_poo
Repositório para o desenvolvimento do trabalho final de Programação Orientada a Objetos, O Sistema Integrado de Gestão de Negócios (SIGNE).

## convenções
Quando forem adicionar algo usem essa convenção:

| Nome da branch           | Significado               |
| ------------------------ | ---------------------- |
| *feature/nome-da-adicao*   | nova funcionalidade    |
| *bugfix/nome-do-erro*      | corrigindo um erro     |
| *doc/assunto-geral*        | atualizar documentação |
| *test/feature-testada*     | criar testes           |
